OPENZERO 5.4

OpenZero is a local-first sovereign operator stack with Hive coordination, optional federation routing, air-gapped friendly operation, self-healing service hooks, and an offline-ready voice layer.

If you are new to the system, start with the Super Panel. It exposes the model, Hive, API key, and voice settings directly so you do not need to hand-edit environment files.
