# Roadmap

This roadmap is a working direction, not a promise of exact release dates.

## Near Term

- More install diagnostics.
- Better first-run onboarding in the Super Panel.
- Cleaner model recommendations by RAM/CPU tier.
- More docs and screenshots.
- More ZeroThink bridge examples.
- Voicebox profile selection improvements.
- Safer reverse proxy examples.

## Mid Term

- Extension manifest loader for private/premium modules.
- Signed extension verification.
- More offline bundle controls.
- More local benchmark reporting.
- Better model download progress.
- Improved Moltbot reliability diagnostics.
- Exportable node health reports.

## Long Term

- Richer multi-node workflows.
- Stronger local/private learning tools.
- Business deployment templates.
- More accessible no-code AI node setup.
- Better local voice/vision workflows.
- Better integrations across TalkToAI, ZeroThink, OpenZero, ZSEC, and FreeWebPanel.

## Principles

- CPU-first should remain real, not decorative.
- Local/private should be the default posture.
- Premium code should stay outside the public repo.
- The public project should be useful on its own.
- Docs should be honest, practical, and beginner-readable.
